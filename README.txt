===========
Stocker
===========

Stocker is opensource framework which allows to easily create 
and benchmark stock exchange investment strategies. 
It includes some real stock exchange data 
to create benchmark simulations. Investment strategies 
can be implemented in python.

How to perform a simulation
=========


Prerequisites
-------------

To perform basic simulation you need:

* Stock exchange data (directory data in Stocker)
* Investment strategy (basic strategies provided in /stocker/SSP/investors/)
